关于定价与权益

Q 免费版能看到什么？
注册后可获得 3 次免费查询。每次查询会展示一个明确的风险点、风险等级摘要和判断依据简版。完整风险点、相似点差异点、判例引用和处理建议在付费版本中提供。
Q 个人版和企业版的核心区别？
个人版面向单人使用，按月订阅，适合个人卖家和高频上新运营。企业版按年订阅，支持多账号、团队协作、风险库管理，适合多店铺、多部门和有 OEM/ODM 业务的制造企业。
Q 报告结果可以作为法律结论吗？
不可以。招财鸟报告用于风险识别和商业决策参考，帮你判断在投入前应该停、改、继续还是申诉。具体的法律行动需结合专业律师意见。
Q 企业定制版具体能定制什么？
包括 API 接入企业内部系统（ERP/PLM/电商后台）、自定义报告模板、多目标市场风险地图、专属知识产权数据库定向接入、战略客户经理 1V1 服务等。具体根据业务需求设计。
Q 点数用完了怎么办？
个人版可购买额外点数包补充；企业版可升级到更高档位；企业定制版默认不限次数。所有套餐都不会"超用扣费"，用完即停，避免账单意外。
Q 数据安全和隐私如何保障？
所有上传资料采用端到端加密传输和存储，仅用于本次查询使用。企业版支持私有化部署选项，敏感新品资料可在企业内部完成检测。
Q 改完款再查会重新扣点数吗？
免费改款入口针对同一产品的修改复查不重复扣点，鼓励用户用招财鸟"查 → 改 → 复查"形成闭环。跨产品查询会按正常点数消耗。
Q 我上传的产品图会被用于训练 AI 模型吗？
绝对不会。你上传的产品图、设计稿、文案、商标词等查询内容永不进入招财鸟的 AI 模型训练集，仅用于本次查询的风险判断。这是写入隐私政策的承诺，详见 安全中心。
Q 上传的资料会保留多久？怎么删除？
免费版上传资料 7 天后自动清除；付费版用户可在账户设置中自定义保留期（最长 1 年）；用户可随时主动一键删除上传资料和历史报告，操作不可恢复。
Q 企业定制版的私有化部署是什么？
大客户可申请把招财鸟部署到独立云环境或客户内部 IDC，敏感新品资料和未公开 IP 不出企业边界，物理资源完全隔离。符合等保三级、数据出境合规和 ISO 27001 要求。
Q 招财鸟有哪些合规认证？
ISO 27001 信息安全管理体系、SOC 2 Type II、网络安全等级保护三级、GDPR 与《个人信息保护法》合规、《生成式人工智能服务管理暂行办法》备案。详见 安全中心。
Q 可以申请试用企业版吗？
可以。联系销售团队可申请 7 天企业版试用，包含 50 次查询权益和团队协作功能体验。点击下方"预约企业方案"提交需求即可。
